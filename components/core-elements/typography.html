CSS and media queries

Font sizes and families are set for all users. The special fonts served via Typekit are included in their own section.

For mobile users the basic font size is increased and other font sizes are enhanced to allow for easier scanning and legibility.

Use of Javascript

The Typekit load script is inserted in the head of the HTML. This injects a script into the page which downloads the appropriate font-face file for the browser in use.
